Preliminary Wrangling¶

FordGoBike data has a size of (183412 x 16) that provides infomation about bike trips during Feb. 2019. Each trip is anonymized and includes:

  • Trip Duration (seconds)
  • Start Time and Date
  • End Time and Date
  • Start Station ID
  • Start Station Name
  • Start Station Latitude
  • Start Station Longitude
  • End Station ID
  • End Station Name
  • End Station Latitude
  • End Station Longitude
  • Bike ID
  • User Type (Subscriber or Customer – “Subscriber” = Member or “Customer” = Casual)

What is the structure of your dataset?¶

FordGoBike data has a size of (183412 x 16) that provides infomation about bike trips during Feb. 2019. Each trip is anonymized and includes:

Trip Duration (seconds) Start Time and Date End Time and Date Start Station ID Start Station Name Start Station Latitude Start Station Longitude End Station ID End Station Name End Station Latitude End Station Longitude Bike ID User Type (Subscriber or Customer – “Subscriber” = Member or “Customer” = Casual)

What is/are the main feature(s) of interest in your dataset?¶

I'm most interested in figuring out the average duration of trips, most common station, which gender, and what age uses the bikes more, and what day includes the highest bike rent. Therefore we can provide this right station with more appropriate bikes that suit the usage purpose.

What features in the dataset do you think will help support your investigation into your feature(s) of interest?¶

I expect that Start Time and Date End Time and Date will help understanding trip duration and days with highest bike demand. In addition Start Station Name and End Station Name will be beneficial in getting most common sations. Finally Member Gender, and age will be helpful in this research.
